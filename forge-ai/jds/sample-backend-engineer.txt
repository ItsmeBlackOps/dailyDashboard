Senior Backend Engineer — Payments Platform
Stripe · Austin, TX (Hybrid)

About the role
We are hiring a Senior Backend Engineer to join the Issuing team. You will own
services that power card issuance, transaction authorization, and dispute
workflows for thousands of businesses. The role is deeply technical and
requires comfort operating distributed, high-throughput systems in production.

What you'll do
- Design and build Java / Spring Boot microservices deployed to AWS (EKS, ECS,
  Lambda) serving 10K+ requests per second
- Own event-driven flows built on Kafka, including exactly-once processing
  patterns and replay tooling
- Work with PostgreSQL at scale — schema design, partitioning, query tuning
- Design and document REST APIs consumed by internal teams and external partners
- Participate in on-call rotation, author runbooks, drive incident post-mortems
- Mentor mid-level engineers, lead design reviews, contribute to platform RFCs

What we're looking for
Must-have:
- 5+ years building production backend services in Java and Spring Boot
- Hands-on AWS experience (EC2, S3, Lambda, SQS/SNS, IAM)
- Production experience with Kafka and event-driven architecture
- PostgreSQL at scale (partitioning, replication, query optimization)
- Kubernetes in production
- REST API design and versioning
- Strong written communication — you can write a design doc that a staff
  engineer can review and a new hire can understand

Nice to have:
- Payments / card networks / ISO 8583 background
- OIDC / OAuth 2.0 / JWT experience
- gRPC
- Observability tooling (Datadog, OpenTelemetry)
- Experience mentoring junior engineers

Culture
We move fast, we write things down, we take ownership. The team is remote-
friendly but meets in Austin monthly. We value engineers who can disagree and
commit, and who prefer boring technology that scales over clever technology
that breaks.
