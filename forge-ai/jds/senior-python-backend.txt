Senior Python Backend Engineer
Airbnb · San Francisco (Hybrid)

About the role
We are hiring a Senior Python Backend Engineer to work on the Trust and Safety
platform. Your services decide — in milliseconds — whether a booking, a review,
or a payment is safe to accept. You will work across ML inference services,
rules engines, and case-management backends used by internal operators.

What you'll do
- Build and scale Python services (FastAPI / async) serving 4K+ RPS of risk decisions
- Integrate ML model inference (TensorFlow Serving, Triton) into the decision path
- Own PostgreSQL schemas for case management, with careful partitioning for retention
- Instrument end-to-end traces with OpenTelemetry + Datadog APM
- Partner with Data Scientists to productionize new model versions weekly
- Lead on-call for a tier-1 service; mentor mid-level engineers

What we're looking for
Must-have:
- 5+ years of production Python experience
- FastAPI or Flask at scale (async patterns, connection pooling, timeouts)
- PostgreSQL schema design and query optimization
- REST API design, versioning, backwards compatibility
- AWS (EC2, RDS, SQS, Lambda, S3)
- Docker + Kubernetes in production
- Strong written communication

Nice to have:
- ML model serving background (TF Serving, Triton, ONNX)
- gRPC
- Risk / Trust & Safety domain
- OpenTelemetry
- Prior mentorship experience

Culture
Slow is smooth, smooth is fast. We document before we code, we prefer reversible
decisions, and we are allergic to premature abstractions. Hybrid 3 days/week in SF.
